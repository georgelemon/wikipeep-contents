# Open to Everything.
WikiPeep is open to contributions, issue reports and anything that can make WikiPeep a better platform.

## Development Guidelines
If you want to contribute to WikiPeep, be sure you fork the latest version [from **main** branch](https://github.com/georgelemon/wikipeep/tree/main), create a branch that represent your changes and make a pull request.

## Issue Reports Guidelines
If you discovered a bug or you have some ideas that can improve the functionality of WikiPeep, be sure you follow some simple guildeline. In order to keep things well organized we need to keep issues separately by their type.

#### Found bugs? [Lets solve them](https://github.com/georgelemon/wikipeep/issues/new?assignees=&labels=&template=bug_report.md&title=)
You don't need to be a developer to report bugs. <mark>Anyone can contribute to WikiPeep</mark>.

#### Got some ideas? [Be creative, let it go!](https://github.com/georgelemon/wikipeep/issues/new?assignees=&labels=&template=feature_request.md&title=)

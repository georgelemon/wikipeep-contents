# How to write my markdown files?
As you already know, WikiPeep does not provide an admin interface so you'll need to write your contents in your favorite IDE or text editor. Personally, I'm using **Sublime Text** editor with a Markdown syntax support.

### Okay, but why Sublime?
I prefer this IDE because it keeps me in the coding mood, without disturbing colors, borders and so on.
